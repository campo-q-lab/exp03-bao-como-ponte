# Experimento 3 – BAO como Ponte  
Inferência do horizonte acústico (r_d) entre medições early e late universe

## 1. Objetivo

Explorar o papel das Oscilações Acústicas de Bárions (BAO) como elemento de ligação
entre o universo primitivo (CMB/Planck) e medições tardias da constante de Hubble (H₀),
avaliando como diferentes valores de H₀ implicam diferentes inferências para o
horizonte acústico (r_d).

O objetivo é investigar se BAO favorece consistentemente o regime early-universe
ou se acomoda valores late-universe sem necessidade de ajustes adicionais.

## 2. Hipótese Testada

A hipótese testada é que a tensão observada em H₀ se manifesta, de forma indireta,
como uma tensão no valor inferido do horizonte acústico quando se tenta compatibilizar
medições late-universe com a estrutura observacional de BAO.

Assume-se que BAO funcione como um “ponteiro estrutural” entre regimes cosmológicos.

## 3. O que o Código Mostra

O código:
- utiliza dados BAO consolidados do SDSS;
- adota uma aproximação simples de escala entre H₀ e r_d;
- infere valores efetivos de r_d compatíveis com H₀ late;
- compara esses valores com o r_d derivado do Planck;
- mostra que:
  - valores SH0ES implicam maior deslocamento de r_d;
  - valores CCHP implicam menor tensão com Planck;
  - BAO tende a favorecer o regime early-universe.

## 4. O que o Código NÃO Mostra

Este código **não**:
- resolve a integral real do horizonte acústico;
- executa ajuste ΛCDM completo;
- inclui parâmetros como N_eff ou energia escura dinâmica;
- fornece explicação causal para a tensão;
- constitui prova da Teoria do Evento.

A inferência de r_d é deliberadamente aproximada.

## 5. Status Epistemológico

Toy model conceitual.

O experimento é propositalmente simplificado para evidenciar tendências estruturais,
não para competir com análises cosmológicas completas.

## 6. Relação com a Teoria do Evento

Este experimento não valida a Teoria do Evento.

Ele é utilizado como apoio ilustrativo para a ideia de que parâmetros globais podem
emergir de regimes distintos de coerência observacional, com BAO atuando como
estrutura de compatibilização entre domínios.
